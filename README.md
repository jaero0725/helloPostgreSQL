# helloPostgreSQL
PostgreSQL 연습 

#### dependency 추가
> 의존성 추가
~~~xml
    <!-- postgreSQL -->
		<dependency>
			<groupId>org.postgresql</groupId>
			<artifactId>postgresql</artifactId>
			<version>42.2.5</version>
		</dependency>
~~~

#### properties 파일

~~~
db.url=jdbc:postgresql://localhost:5432/testdb
db.user=postgres
db.passwd= "공백"
~~~
